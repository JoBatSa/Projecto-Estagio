
Em backend criação de base de dados de forma a armazenar de forma persistente toda a informação e aplicação para  gestão.  Em  frontend  criação  de  web  api  para  acesso  através  de  autenticação  pelos  funcionários  (auditores, administrativos, ETC), de forma a poder ser acedida quer por computador, tablet ou smartphone. 1. Através de uma web api introduzir diretamente os dados necessários e gerando automaticamente um relatório que será enviado no fim do mesmo serviço ao cliente. 2. Os auditores só poderão iniciar um serviço depois de terem formação e um supervisor lhes der autorização para tal e sempre que tiverem alguma atualização serão informados através da aplicação por uma gama visual. 3.  Automatização  de  forma  que  sempre  que  seja  iniciado  um  novo  serviço  e  verificado  que  o  auditor  tem autorização para o fazer ser atribuído automaticamente o “work order”
